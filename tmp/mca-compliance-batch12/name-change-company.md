## Name change of company

### Detailed overview

Change of company name is governed by Section 13 of the Companies Act, 2013, requiring shareholder approval through special resolution and Central Government (in practice, MCA) approval. Name changes are common during rebranding, business pivot, M&A, or strategic positioning.

The process involves name availability check via RUN (Reserve Unique Name), shareholder approval, MoA amendment, and Central Government approval, followed by issuance of new Certificate of Incorporation.

### Key features

- RUN reservation, special resolution, INC-24 and MGT-14
- New COI and then PAN, GST, bank, and contract updates
- Name guidelines and availability checks reduce rejection risk

## Applicability

All companies, private, public, OPC, Section 8. Pan-India.

## Compliance requirements

| Step | Requirement |
| Name reservation | RUN application, ₹1,000 fee |
| Shareholder approval | Special resolution (75%) |
| MoA amendment | New name in MoA |
| Forms | INC-24 (application for name change) and MGT-14 (special resolution) |
| Central Government approval | Effectively granted by RoC |
| New COI | Fresh Certificate of Incorporation issued |

## Process

1. Apply for name approval via RUN (₹1,000 fee)
2. Once approved (valid for 20 days), board approves name change
3. EGM notice (21 days)
4. EGM passes special resolution
5. File INC-24 and MGT-14 with MCA within 30 days
6. RoC reviews and issues new COI
7. Update PAN, GST, bank accounts, signage, etc.

## Timeline

30 to 60 days end-to-end.

## Frequently asked questions

**1. Are there restrictions on company names?** Yes. Names cannot be: identical or closely resembling existing companies, using restricted words (National, Bharat, Government without permission), offensive, misleading. MCA name guidelines must be followed.

**2. What if RUN application is rejected?** Reapply with alternative names. Up to 2 names can be proposed in a single application. Reasons for rejection are usually similarity or restricted words.

**3. Does name change affect the company's PAN?** Yes, PAN must be updated with the new name. Apply for PAN correction with new COI.

**4. What about pending contracts and bank accounts?** All contracts continue (the company is the same legal entity, just renamed). Bank accounts, GST, and contractual relationships need to be updated with the new name. Some banks may issue fresh accounts, others update existing.

**5. Can a company change its name multiple times?** Yes. There is no statutory limit on number of name changes. However, frequent changes can raise stakeholder concerns and have practical costs.
