# Temporary One Day Function License

## Short overview

A Temporary One Day Function License is a comprehensive single-day event license that combines liquor service permission with other event-related approvals (food service, entertainment, public assembly) for organised functions like weddings, corporate events, large social gatherings, and public functions. While "One Day Permission for Foreign Liquor" focuses specifically on liquor service, the One Day Function License is often a broader composite permission covering multiple regulatory aspects of large events.

The license enables event hosts and organisers to legally conduct large-scale events with full compliance, covering liquor service, food service, fire safety, public assembly, and entertainment, through a streamlined single-window approval (in states with such systems) or coordinated multi-department approvals.

## Key features

- Single-day comprehensive permission
- Multi-department coordination
- Event-specific conditions
- Quantity-based liquor permission
- Capacity-based fire and safety compliance

## Benefits

- Single-window approval for multiple event-related permissions in some states
- Comprehensive coverage for liquor, food, entertainment
- Reduces multiple agency follow-ups
- Suitable for large weddings (500+ guests)
- Corporate events, product launches, conferences

## Limitations

- Single day only
- Higher fees than basic one-day liquor permission
- Strict event-specific conditions
- Heavy compliance requirements
- Last-minute approvals difficult

## Applicability

Available in most states with varying nomenclature: Maharashtra (One Day Function License, composite), Karnataka (Special Event Permit), Delhi (Composite Event Permission), Tamil Nadu (Special Function License), Telangana (Event Function License).

## Process

Apply 30 to 45 days before event with comprehensive documentation. Multi-department clearance and approval. Pay consolidated fees. Receive license. Conduct event with all conditions met.

## Validity and renewal

Single day only. No renewal, fresh application for each event.

## Frequently asked questions

**1. Is one composite license enough or do I still need separate permissions?** In some states (Maharashtra, Delhi), composite licenses cover most needs. In other states, separate permissions for liquor, food, entertainment may still be needed.

**2. Can the function be held in a residential building?** Possible but more complex, requires building society NOC, parking arrangements, and noise compliance. Better to use commercial venues.

**3. What if attendance exceeds the declared number?** This is a violation. Apply for a higher capacity in the original application or seek upward revision before the event.

**4. Is event insurance mandatory?** Not legally mandatory in most states, but strongly recommended. Some venues require it.

**5. Can DJ or live performance be conducted under this license?** Entertainment elements require additional Public Performance License (PPL) and Police Entertainment License. The function license alone may not cover music or performance.
