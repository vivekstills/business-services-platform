# RERA Registration for Projects

## Short overview

RERA Project Registration is mandatory under the Real Estate (Regulation and Development) Act, 2016 for any real estate project, residential or commercial, exceeding 500 sq.m. or 8 apartments. The promoter (developer or builder) must register the project with the State RERA Authority before advertising, marketing, booking, selling, or offering for sale any apartment, plot, or building in the project.

Project registration is a foundational consumer protection mechanism under RERA, ensuring transparency in project details, timelines, financial accounts (70% of buyer money in escrow), and accountability for delivery commitments.

## Key features

- Project-specific registration
- Phase-wise registration permitted
- Project-completion validity
- 70% escrow account mandatory
- Quarterly compliance reporting

## Benefits

- Legal authorisation to launch and market project
- Buyer trust and project credibility
- Eligibility to receive booking amounts and advances
- Access to construction finance from banks or NBFCs
- Protection from suits for unregistered project sales

## Limitations

- Significant documentation and compliance
- 70% escrow account requirement
- Quarterly project update obligations
- Penalties for delays and misrepresentation
- Annual audit requirement
- Construction strictly per approved plans

## Applicability

Mandatory in all states with active RERA. Maharashtra (MahaRERA) leads in stringent enforcement.

## Documents required

| Document | Details |
| --- | --- |
| Application form | RERA Form A or state equivalent |
| Promoter details | PAN, identity, address proofs |
| Project details | Land details, plan, approvals |
| Land title documents | Title deed, mutation, encumbrance |
| Sanctioned plans | Building plan, layout, approvals |
| All government approvals | Environment, fire, height clearances |
| Architect details | Registered architect appointment |
| Engineer details | Structural engineer details |
| Estimated project cost | Detailed cost projection |
| Estimated timeline | Phased completion schedule |
| Escrow account proof | Bank account opening proof |
| Past project disclosure | Promoter's past 5 projects |

## Process

1. Obtain all preliminary approvals (land, planning, environment).
2. Open separate project escrow account.
3. Apply on State RERA portal.
4. Upload documents and project plans.
5. Pay registration fees.
6. RERA Authority review.
7. Project registration with unique RERA number.
8. Begin marketing and bookings.

Timeline: 30 to 90 days for clean applications.

## Validity and renewal

Valid until project completion as per registered timeline. Extension applications possible with valid justification. Failure to complete on time attracts penalties.

## Frequently asked questions

**1. Can construction begin before RERA registration?** RERA registration must precede marketing and sales but construction can start with all other approvals. However, sales are prohibited without RERA registration.

**2. What happens if a project is not completed on time?** Promoter must compensate buyers with interest (typically MCLR + 2%) on delayed possession. Continued delays attract de-registration and penalties.

**3. Is the 70% escrow mandatory for all projects?** Yes. 70% of amounts collected from buyers must be deposited in the project-specific escrow account, withdrawable only for project construction expenses certified by architect or engineer.

**4. Can a single project be registered in phases?** Yes. Large projects can be registered phase-wise, with each phase having its own RERA number and compliance.

**5. Do small builder projects below threshold need RERA?** Projects below 500 sq.m. or 8 units are exempt from project registration but agents dealing in such projects still need agent registration.
